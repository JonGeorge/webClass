Update your Maven Plugin for Eclipse

1) Startup Eclipse
2) Pull Help / Install New Software....
3) Next to "Work With", paste-in this URL
   http://download.eclipse.org/technology/m2e/releases
   
4) Check the checkbox next to Maven Integration for Eclipse
     -- Select    m2e - Maven Integration for Eclipse
	 -- Select    m2e - slf4j over logback logging
	 
   Press <Next>
   
5) You will get a warning about stuff that has to be upgraded
   Press <Next>
   
6) In the "Review Licenses"
   Select "I accept the terms..." and press <Finish>
   
7) You will be prompted to restart Eclipse
   Press <Yes>
