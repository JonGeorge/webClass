How to Setup a Unit Test with a Spring Data Source
--------------------------------------------------

Assumptions:
 A) You want to run a unit test
 B) You are using the Spring Framework
 
 
Procedures
----------

1. Setup a DataSource using a Tomcat JDBC Connection Pool
   

2. Create a DAO class that has a DataSource injected


3. Create a unit test that tests the Dao class


4. Run the unit tests in Eclipse
