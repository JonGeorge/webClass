How to make LibreOffice Calc Run Faster
---------------------------------------
 1. Enable OPENGL
    Pull Tools / Options / Libre Office / View / 
      Check "Use OpenGL For all rendering"

 2. Uncheck use anti-aliasing
     Pull Tools / Options / Libre Office / View / 
       Uncheck Use "anti-aliasing"
       Uncheck "Screen font antialiasing"
       
 3. Increase memory
    Pull Tools / Options / LibreOffice / Memory 
      Use for LibreOffice:  128 MB
      Memory per object:     20 MB
 
 4. Uncheck all of the VBA properties
    Pull Tools / Options / Load/Save / VBA Properties
      Uncheck all entries
