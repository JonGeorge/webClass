 1. What is HTML?
    HTML is the standard markup language for Web pages
    HTML elements are the building blocks of HTML pages
    Your web browser recognizes HTML tags -- so the web browser is your partner

    <html>

    <body>

    <form method="POST" action="http://localhost:8080/somewhere">

      <input type="text" name="customer_name"></input>
      <input type="password" name="customer_password"></input>
      <button type="submit" name="btnLogin">Login</button>

    </form>

    </body>
    </html>


 2. History of web programming
    1990s:  apache/perl or apache/php
            HTML 3


    2000s:  ASP, JSP
            -- The server is rendering the page
            -- You make calls ASP or JSP pages and they return entire pages or parts of pages

    2000s:  AJAX:  Asynchronous JavaScript and XML
            -- Make a call to the backend and *not* wait

    2000s:  jQuery
            -- Powerful javascript framework
            -- Turns your code into speghetti

    2010s:  AngularJS arrives
            -- The first framework that makes it easy to maintain webapps
            -- Allows you to create your own tags (directives)
            -- Makes it easy to set fields
            Node, Node Package Manager

    2017:   Angular (next generation)
