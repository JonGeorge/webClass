How to Run Schema Spy to Generate an Database Diagram as an Intranet
--------------------------------------------------------------------

Assumptions:
 A) You have a mysql database
 B) You want to generate the table schema
 C) You know the mysql schema username/password


Procedures
----------
 1. Install GraphViz 2.28 library
 
 2. Download SchemaSpy jar
 
 3. Download the mysql connector jar
 
 4. Create DOS batch script that will
     a) Add graphviz 2.28\bin to your PATH
     b) delete the destination directory
     c) Run java -jar schemaspy_5.jar -t mysql -u <mysql username> -p <password -o "%DEST_DIRECTORY%" -host <hostname> -db <database name> -port 3306 -cp mysql-connector-5.1.6.jar
