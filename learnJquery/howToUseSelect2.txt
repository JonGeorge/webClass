How to Use Select2 for your dropdowns (with typeahead)
------------------------------------------------------

1) Download the select2 package

2) Add the select2 package to your web app

3) Make sure your web page includes the select2.jss and select2.css files

4) Setup a <select tag>
     <html>
        <body>
           This is my select tag<br/>
           <select id="myDropdown">
               <!-- Include an empty dropdown if you want to use the placeholder -->
               <option></option>
               <option value="MD">Maryland</option>
               <option value="DC">District of Columbia</option>
               <option value="VA">Virginia</option>
           </select>
        </body>
     </html>
     
5) Initialize it in your jquery code
    var myDropdown = $('#myDropdown');
    myDropdown.select( {
       placeholder:  "-Select a value-"
       });
