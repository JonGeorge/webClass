How to Install DosBox on Centos 7
---------------------------------

References
----------
https://it.megocollector.com/linux/install-dosbox-on-centos-7/


Procedure
---------
 unix> sudo yum install -y https://schotty.com/yum/el/7/schotty-el7-release-7-1.noarch.rpm
 unix> sudo yum install -y http://dl.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-5.noarch.rpm
 unix> sudo yum install -y http://copr-be.cloud.fedoraproject.org/results/tomkep/epel-extras/epel-7-x86_64/epel-extras-7-1/epel-extras-7-1.noarch.rpm
 unix> sudo yum install dosbox
