How to Install Git and use with Intellij
----------------------------------------

Procedures
----------
 1. Download and install Github.exe
    NOTE:  This installs a portable git here:
      %USERPROFILE%\AppData\Local\GitHub\PORTAB~1\cmd\git.exe
      
 2. Tell Intellij where to find git.exe
    a. Startup Intellij
    b. Close any existing project
    c. Pull Configure -> Settings
    d. Browser to Version Control -> Git
       Path to GIT executable:  C:\Users\ <your windows login> \AppData\Local\GitHub\PORTAB~1\cmd\git.exe
  
    e. Press the "Test" button
       -- It should say 
          Git Executed Successfully
