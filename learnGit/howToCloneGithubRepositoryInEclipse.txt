How to Clone a GitHub Repository using Eclipse
----------------------------------------------

Assumptions
 A) You have git.exe installed  [see howToInstallGitOnWindows.txt]
 B) You have Intellij installed
 C) You want to clone an existing repository located here:
      https://github.com/traderres/webapps.git
 D) You want to store this project locally in this directory:
      c:\tools\eclipse\workspace\webapps
 
Procedures
----------
 1. Startup Eclipse
 
 2. Pull File -> Import
    a. Select Git -> Projects from Git
       Press Next
    
    b. Select "Clone URI"
       Press Next
       
    c. In the "Source Git Repository" page
       URI:  https://github.com/traderres/webapps.git
       
       Authentication:    <Your github username>
       Password:          <Your github password>
       Check "Store in Secure Store"
       Press Next
       
    d. In the "Branch Selection" page
       Check "Master"
       Press Next
    
    e. In the "Local Destination" page
       Directory:      c:\tools\eclipse\workspace\webapps
       Initial branch: master
       Press Next
     
       Now, Eclipse will clone the project
     
    f. Press the "Back" button
    g. In the "Local Destination" page
       Press "Next"  [yes, press it again!]
    
       Now, the "Select a wizard" page shows the Working directory 
       
    h. In the "Select a wizard to use for importing projects"
       Check "Import as general project"
       Press Finish
    
    h. In the "Import Project"
       Project name:  webapps   
       Press Finish
    
 3. Verify that git works by viewing the history
    Right-click on webapps -> Team -> Show in history
