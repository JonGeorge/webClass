Learn Spring MVC REST Endpoint Parameters
-----------------------------------------


Lessons
-------
 1. Create a Model Class that holds your Search DTO object


    public class SearchDTO  {
      private String rawQuery;
      private String indexName;

      public String getIndexName() {
        return this.indexName;
      }

      public void setIndexName(String indexName) {
        this.indexName = indexName;
      }

      public String getRawQuery () {
        return this.rawQuery;
      }

      public void setRawQuery ( String rawQuery ) {
        this.rawQuery = rawQuery;
      }

    }


  2. Pass-in an entire Object (using request body)

     /*
      * /api/search
      * {
      *     index: "this",
      *     rawQuery: "that"
      * }
      */
     @GetMapping("/api/search")
     public  ResponseEntity<?>  getStuff(@RequestBody SearchDTO searchDto) {
         logger.debug("searchDto={}", searchDto);

         String returnValue = "searchDto=" + searchDto;

         // Return the string as plain-text
         return ResponseEntity
                 .status(HttpStatus.OK)
                 .contentType(MediaType.TEXT_PLAIN)
                 .body(returnValue);
     }
