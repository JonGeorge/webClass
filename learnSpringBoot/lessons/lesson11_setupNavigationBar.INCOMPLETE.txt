Lesson:  Setup a Navigation Bar (using Angular UI Router)
---------------------------------------------------------
Our web application eneds a real navigation bar and its own navigation controller

In this lesson, we will setup a basic look & Feel with a navigation bar on the left side
    We will include these options:
       Main
         Dashboard
         Search Page
         User Profile

       Admin
         Admin Page1
         Admin Page2


Procedure
---------
 1. Add Angular material to your app.html page

 2. Setup Look & feel

 3. Create this file:  navigation.controller.js

 4. Adjust app.html to with ui-sref="page1" for the links
