How to get RGB Coordinates of a point on an image using GIMP
------------------------------------------------------------

Assumptions
 A) You have GIMP installed
 B) You have the image that has a color or point of interest
 
Procedure
---------
 1. Open the image with the Gnu Image Manipulation Program (GIMP)

 2. Pull Window -> New Toolbox  (or press Control-B)

 3. In the Toolbox, Select the Color Picker Tool (or press O)

 4. Click on the point in the image

 5. In the Toolobx, select the Foreground/Background colors icon (bottom icon)

 6. In the "Change Foreground Color" popup
    You will see HTML notation
