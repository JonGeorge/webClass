How to Make Remote Git Branch
-----------------------------


Procedures
----------
 1. Checkout the project
 2. Create a new local banch
    Pull VCS -> Git -> Branches...
    Single-click on origin/master -> Checkout as new local branch
    
    In the "Checkout Remote Branch" box
      Enter name of branch:  private/aresnick/something
      Press OK
    
 3. Now push the local branch to the remote site
    Pull VC -> Git -> Push...
    Single click on the "master" hyperlink
    Type-in the name of your remote branch:   private/aresnick/something
    Press "Push"
