How to Tell Intellij to Not Open Last Project
---------------------------------------------

Procedure
---------
 1. Startup Intellij
 2. If a project opens, pull File / Close Project
 3. In the "Welcome to Intellij IDEA" window, Pull Configure -> Settings
 4. Browse to Appearance & Behavior -> System Settings
 5. Uncheck "Reopen last project on startup
 6. Press Apply 
 7. Press OK
