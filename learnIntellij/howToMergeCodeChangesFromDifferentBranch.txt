How to Merge Code Changes from a Different Branch (or Trunk) in Intellij
------------------------------------------------------------------------

Assumptions:
 A) You have Intellij
 B) You have a subversion repository
 C) You want to merge-in code changes from a branch to trunk
 
 
Procedure
---------
 1. Open your project  [it should be the trunk]
 2. Update to HEAD
 3. Click on the "Changes" tab
 4. Click on "Subversion Working Copies Information"
 5. Click on "Merge From..."
    a. Press "Branches...."
    b. Select the branch to merge from
    c. Click "Quick Manual Select'
    d. Check-off your changes you want to merge
