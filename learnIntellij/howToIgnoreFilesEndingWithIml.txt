How to Tell IntelliJ to ignore all .iml files (for Version Control purposes)
----------------------------------------------------------------------------

Problem:  You have lots of .iml files in your Changes window but you will not check in these files

Solution:  Tell Intellij to ignore it:


Procedure
---------
 1) Pull File / Settings
 2) Version Control -> Ignored Files
 3) Select "Ignore all files matching"  *.iml
    Press OK
