How to Install Python 3 on Windows 7
------------------------------------

Assumptions
 A) You have a Windows 7 or later computer
 B) We will install the 32-bit version of Python (because that version has greater library support)
    NOTE:  If you truly want 64-bit version of Python, then download 
           https://www.python.org/ftp/python/3.5.1/python-3.5.1-amd64.exe
           

Procedure
---------
 1. Download Python 3.5.1 (32-bit) 
    a. Go to https://www.python.org/downloads/windows/
    b. Click on "Latest Python 3 Release - Python 3.5.1"
    c. Scroll down to the "Files" section on the bottom
    d. Click on "Windows x86 executable installer"
       *OR*
       Go to https://www.python.org/ftp/python/3.5.1/python-3.5.1.exe
    
    e. Save python-3.5.1.exe to your c:\vault\
  
  
 2. Install Python 3.5.1 for Windows (32-bit) version
    a. Run c:\vault\python-3.5.1.exe
    b. In the "Install Python 3.5.1 (32-bit)" screen, press "Customize installation"
    c. In the "Optional Features" screen, go with the defaults (check everything) and press Next
    d. In the "Advanced Options",
       Check "Associate files with Python"
       Check "Create shortcuts for installed applications"
       
       Next to Customize install location,
          C:\tools\Python35-32
       
       Press "Install"


 3. Verify that Python works
    a. Open a DOS window by pressing <Start><Run>CMD
       CMD> py --version
            Python 3.5.1
            

 4. Open notepad
    a. Copy this to notepad
       total=5
       print("total is ", total)       
    
    b. Save this to c:\temp\intro.py
    
    c. Run your python script
       CMD> py c:\temp\intro.py    
       total is  5
