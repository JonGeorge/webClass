How to Get Exif Picture Metadata
--------------------------------
ExifRead is a python package that lets you examine the metadata from a picture.
When you take a picture [with a smartphone], the camera apps leave lots of metadata inside the picture


Assumptions:
 A) You have Python 3.4 installed (32-bit)
 B) You have PyCharm installed 
 C) You have a picture
 D) You want extract the Exit picture metadata from that picture
 
 
Procedures
----------
 1. Verify that you have pip
    a. Open a DOS window by pressing <Start><Run>CMD
    b. In the DOS window, type-in this:
       DOS> pip --version
       
       You should see this:
         pip 7.1.2 from c:\tools\python34-32\lib\site-packages (python 3.4)


  
 2. Install ExifRead using pip
    a. Open a DOS window by pressing <Start><Run>CMD
    b. In the DOS window, type-in this:
       DOS> pip install ExifRead
    
      You should see this:
        Collecting ExifRead
        Downloading ExifRead-2.1.2.tar.gz
        Installing collected packages: ExifRead
        Running setup.py install for ExifRead
        Successfully installed ExifRead-2.1.2

    NOTE:  You should see the files installed to here:
          %PYTHON_HOME%\Lib\site-packages\exifread\


 3. Create the listPictureMetadata.py 
    a. Startup PyCharm
    b. Right-click on project1 -> New -> Python File
       Name:  listPictureMetadata.py
       
    c. Copy this to listPictureMetadata.py
       WARNING:  Do not add any leading spaces
        
        #######################################################################
        # Filename:  listPictureMetadata.py
        # Author:    <your name>
        #######################################################################
        # Purpose:
        #   List all of the metadata from a picture
        #
        #  Design:
        #    1) Verify that the user passed-in all of the required parameters
        #
        # Usage
        #   DOS> python listPictureMetadata.py --file="somePicture.jpg" <enter>
        #
        # Assumptions
        #   A) python is in the PATH
        #######################################################################
        import os
        import time
        import sys
        import argparse
        import exifread
        
        
        #######################################################################
        # verifyCommandLineArguments()
        #######################################################################
        def verifyCommandLineArguments(aArgs):
            print("verifyCommandLineArguments()")
        
            if not aArgs:
                print("Error:  No arguments were provided")
                sys.exit(1)
        
            elif not aArgs.file:
                print("Error:  No --file argument was provided")
                sys.exit(1)
        
            if (os.path.isfile(aArgs.file) == False):
                print("Error:  I could not find this file:  %s" % aArgs.file)
                sys.exit(1)
        
            if (os.access(aArgs.file, os.R_OK) == False):
                print("Error:  I found this file, but I cannot read it:  %s" % aArgs.file)
                sys.exit(1)
        
            print("Command-line args are valid\n")
            return
        
        # P R O G R A M         S T A R T S        H E R E
        gsScriptName = os.path.basename(__file__)
        gsCWD = os.getcwd()
        
        # Print the date/time when this script started
        print("%s has started as of %s" % (gsScriptName, time.strftime("%c")))
        
        
        # Parse Command-line arguments
        parser = argparse.ArgumentParser(prog=gsScriptName)
        parser.add_argument('--file')
        goCommandLineArgs = parser.parse_args()
        
        # Verify Command-line arguments
        verifyCommandLineArguments(goCommandLineArgs)
        
        # Open the files for binary reading
        fPictureFile = open(goCommandLineArgs.file, 'rb')
        
        # Get a map of tags
        tags = exifread.process_file(fPictureFile)
        
        # Close the picture
        fPictureFile.close()
        
        # Print-out the tags in sorted key order
        for key in sorted(tags.keys()):
            print("%-40s : %s" % (key, tags[key]))
        
        # Print the date/time when this script finished
        print("\n%s has finished as of %s" % (gsScriptName, time.strftime("%c")))
        
        # Python script ends here
        sys.exit(0)
