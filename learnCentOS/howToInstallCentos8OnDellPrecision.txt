How to Install Centos 8 on Dell Precision
-----------------------------------------
NOTE:  Secure boot does not matter.  Centos 8 works either way

Procedure
---------
 1. Create a USB drive with the centos 8 distribution
 2. Change the System configuration to use AHCI
    Press F2
    Click on System Configuration
    Click on Sata Operation
    -- Set it to AHCI
    
 3. When you see the Centos 8 Install menu
    -- Select Troubleshooting
    -- Select Basic Installation
