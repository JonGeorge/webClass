How to Get the Teams Linux Client to work with Wayland
------------------------------------------------------
Problem:  I run CentOS 8 using the default Wayland window manager 
          I open Teams
          I attempt to share my screen
          -- Teams Crashes!
          

Workaround #1: Login using Standard (Xorg)
          -- In the login screen
          -- Enter your username
          -- On the password screen, select the settings
          -- Choose Xorg
