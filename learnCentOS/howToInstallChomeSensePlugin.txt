How to Install the Sense Plugin for Chrome
------------------------------------------
Problem:  Google no longer lets users download the "Sense" plugin

Solution: I uploaded my google chrome extension
          Now, you can download the crx file and install it.
      
WARNING:  This only works on older versions of Chrome
          The Chrome 67 version RPM can be downloaded here:
             https://github.com/traderres/webClass/raw/master/learnCentOS/google-chrome-stable_current_67_x86_64.rpm
      
Procedure
---------
 1. Configure Chrome to not protect you from dangerious sites
    a. Startup Chrome
    b. Go to chrome://settings/privacy
    c. Uncheck  "Protect you and your device from dangerous sites"
    d. Close up Chrome


 2. Download the packed extension to your computer
    a. Open a browser
    b. Go to github.com/traderres
    c. Click on WebClass
    d. Click on chrome_sense_094.crx
    e. Click on Raw
       or
       Go to https://github.com/traderres/webClass/raw/master/learnCentOS/chrome_sense_094.crx
    f. Save it to your Downloads/ directory
    

 3. Install the Extension onto your Chrome environment
    a. Startup Chrome
    b. Go to chrome://extensions/
    c. Check-off "Developer Mode"
    d. Drag & Drop the .crx file into your Extensions
    e. Close up Chrome
    
    
 4. Verify it works
    a. Startup Chrome
    b. Click on the Sense Plugin
       -- You should see the Sense plugin
       
 NOTE:  
   You will get prompted to "Disable Developer mode extensions"
   Press "Cancel"
