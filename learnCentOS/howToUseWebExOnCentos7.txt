How to Use WebEx with Firefox or Chrome
---------------------------------------
Problem:  I want to connect to these WebEx meetings but my browser does not support it
Solution1: Use Firefox and "Cisco Webex Extension"
Solution2: Use Chrome and "Cisco Webex Extension"

NOTE:
  -- These are not *plugins*
  -- You would be installing *extensions*
  -- If you have the webex plugin, I would uninstall it and use the "extension" instead



Configure Firefox with the "Cisco Webex Extension"
---------------------------------------------
ASSUMPTION:  You are installing the auto-updated version of Firefox
             [see learnCentOS / howToInstallFirefoxAutoUpdate.txt]

 1. Startup Firefox
 2. Right-click on the top bar -> Menu Bar

 3. Pull Tools -> Add-ons
    a. Click "Extensions"
    b. In the Find more extensionn, type-in this:
        cisco webex extension
        Press <Enter>
    c. Click on Cisco Webex Extension
    d. Press the Install button



Configure Chrome with the "Cisco Webex Extension"
--------------------------------------------------
ASSUMPTION:  You have installed Chrome for Centos Linux
             [see learnCentOS / howToInstallChromeOnCentos7.txt]

 1. Startup Chrome
 2. Go to extensions page
    chrome://extensions/
 3. In the "Search extensions" box type-in this:
        Cisco Webex Extension
        Press <Enter>

    *OR*

    Go to https://chrome.google.com/webstore/detail/cisco-webex-extension/jlhmfgmfgeifomenelglieieghnjghma?hl=en

 4. Click on "Cisco Webex Extension" and press Install
 5. Press "Add"

 NOTE:  When connecting to WebEx, you will need to click on the "Cisco Webex Extension" -> Allow extension to share Camera/microphone


Troubleshooting
---------------
Problem:  I cannot share my screen with someone else using Chrome
Solution: End the meeting, Open a new Incognito Window in Chrome, Start a new meeting
