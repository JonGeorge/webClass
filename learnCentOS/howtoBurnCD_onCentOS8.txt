How to Burn a CD on CentOS 8
----------------------------


Procedure
---------
 1. Install brasero
    unix> sudo dnf install brasero
    
 2. Run brasero
    unix> brasero
    
 3. Select "Data Project"  (to create a data CD/DVD)
 
 4. Add files to the CD/DVD
    a. Press the +
    b. Find the files to add
    
 5. Press "Burn..."
