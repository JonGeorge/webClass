How to Burn a CD on CentOS 8
----------------------------


Procedure
---------
 1. Install brasero
    unix> sudo dnf install brasero
 
 2. Add a shortcut
    a. Right-click on your panel -> Add to Panel....
	b. Select Custom Application Launcher and press "Add"
	c. In the Create Launcher popup
	   Name:     brasero
	   Command:  brasero
	   press OK
	   

 3. Run brasero
    unix> brasero
    
 4. Select "Data Project"  (to create a data CD/DVD)
 
 5. Add files to the CD/DVD
    a. Press the +
    b. Find the files to add
    
 6. Press "Burn..."
