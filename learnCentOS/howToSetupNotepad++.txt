How to Setup Notepad++ for CentOS
---------------------------------
Compile and Install Notepadq (it's a replica of Notepad++ for Linux)

Assumptions:
 A) You are running CentOS
 B) You want to setup a text editor that looks nearly identical to Notepad++
 


Procedure
---------
 1. Download & Install Qt
    unix> sudo yum install qt5-qtsvg-devel qt5-qtwebkit-devel qt-creator qtchooser qt5-qtwebkit qt5-qtsvg 
          -- Now, you should have qmake located in /usr/lib64/qt5/bin/qmake
          
       
 2. Clone the project
    NOTE:  notepadqq depends on qt 5.3 or greater
    unix> cd ~/Downloads
    unix> git clone https://github.com/notepadqq/notepadqq.git
    
    
 3. Build the project
    unix> cd notepadqq
    unix> ./configure --prefix /usr --qmake /usr/lib64/qt5/bin/qmake
    unix> make
    unix> sudo make install            # You need to be root to install
