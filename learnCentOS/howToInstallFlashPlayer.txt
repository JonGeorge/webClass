How to Install the Adobe Flash Player on Centos
-----------------------------------------------
WARNING:  Flash will always be a source of security holes

Procedure
---------
 1. Startup Firefox

 2. Go to https://get.adobe.com/flashplayer/

 3. It should identify your system as Linux 64-bit, English, Firefox
    a. Select ".rpm for Linux"
    b. Press "Download now"
    c. Save flash-player-npapi-27.0.0.183-release.x86_64.rpm to your ~/Downloads/ directory

 4. Run the rpm file
    a. Open a terminal
    b. sudo yum install ~/Downloads/flash-player-npapi-27.0.0.183-release.x86_64.rpm

 5. Adjust Firefox to prompt you before starting Flash
    a. In Firefox, pull Tools / Add-ons
    b. Select "Plugins"
    c. Next to "Shockwave Flash" change the option to "Ask to Activate"
