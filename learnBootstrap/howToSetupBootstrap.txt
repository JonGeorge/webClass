How to Setup Bootstrap 3.3.4
----------------------------

1) Download Bootstrap
    a) Go to https://github.com/twbs/bootstrap
    b) Scroll down and click on the link that says "Download the latest release."
       -- It should take you to something like https://github.com/twbs/bootstrap/archive/v3.3.4.zip
	   -- Save bootstrap-3.3.4.zip to your c:\vault\ directory
	   
	c) Unzip your boostrap distribution
	   -- Right-click on c:\vault\bootstrap-3.3.4.zip -> 7-zip -> Extract Here
	   -- Now, you should have these directories and more

          C:\vault\bootstrap-3.3.4\
                dist/
                +-- css/
                ¦   +-- bootstrap.css
                ¦   +-- bootstrap.css.map
                ¦   +-- bootstrap.min.css
                ¦   +-- bootstrap-theme.css
                ¦   +-- bootstrap-theme.css.map
                ¦   +-- bootstrap-theme.min.css
                +-- js/
                ¦   +-- bootstrap.js
                ¦   +-- bootstrap.min.js
                +-- fonts/
                    +-- glyphicons-halflings-regular.eot
                    +-- glyphicons-halflings-regular.svg
                    +-- glyphicons-halflings-regular.ttf
                    +-- glyphicons-halflings-regular.woff
                    +-- glyphicons-halflings-regular.woff2
  	      
	     
2) Install bootstrap into your webapp
    a) Startup Eclipse
    b) Right-click on WAR -> new -> folder
	   Folder name:  bootstrap-3.3.4
		   
    d) Browse to the the bootstrap-3.3.4/dist directory to WAR/bootstrap-3.3.4
	   Press <Start><Run>c:\vault\bootstrap-3.3.4
	   Drag & Drop the dist to your Eclipse WAR / bootstrap-3.3.4 
	   Eclipse will prompt you to Copy or Link. 
	   -- Select Copy and press OK
	   
	   -- Now, you should have a WAR / bootstrap-3.3.4 / dist 
	   

3) Setup html5.shiv   
    a) Download html5.shiv
		1) Go to https://github.com/aFarkas/html5shiv
		2) Click "Download Zip"
		   -- Save html5shiv-master.zip to your c:\vault
		
		3) Unzip the c:\vault\html5shiv-master.zip
		   Right-click on c:\vault\html5shiv-master.zip -> 7-zip -> Extract here
		   
		   Now, you should have this directory: C:\vault\html5shiv-master\dist\

    b) Install html5.shiv		     
		1) In Eclipse, right-click on bootstrap-3-3-4 -> new -> folder
		   Folder name: assets
		
		2) Drag & Drop all files from C:\vault\html5shiv-master\dist\  to WAR -> bootstrap-3.3.4 ->  assets 
		
	
	    Now, under the bootstrap-3.3.4 directory in your project, you should see these directories
	           dist / css
	           dist / fonts
	           dist / js
	           assets / 
	          
	
	
4) Setup respond.min.js
    a) Download respond.min.js
		1) Go to https://github.com/scottjehl/Respond
		2) Click "Download Zip" 
		   -- Save Respond-master.zip to your c:\vault\
		   
		3) Unzip c:\vault\Respond-master.zip 
		   Right-click on c:\vault\Respond-master.zip -> 7-zip -> Extract here
		   
 
	b) Install respond.min.js
	    a) Browse to C:\vault\Respond-master\dest\
		b) Drag-and-drop respond.min.js to      WAR/bootstrap-3.3.4/assets

	

5) Setup jQuery in your web app
    a) In your c:\vault, create a directory called jquery
    b) Download jquery 1.11.2 files
      
		1) Go to http://jquery.com/download/#Current_Release
		2) In the jQuery 1.x section
		    --You will see a link that says something like "Download the compressed, production jQuery 1.11.3"
		      -- Right-click on that link and select "Save Link As..."
		      -- You will be prompted to save jquery-1.11.3.min.js
		      -- Save the file to your c:\vault\jquery\
		    
            --You will see a link that says something like "Download the uncompressed, development jQuery 1.11.3"
              -- Right-click on that link and select "Save Link As..."
              -- You will be prompted to save jquery-1.11.3.js
              -- Save the file to your c:\vault\jquery\

            --You will see a link that says something like "Download the map file for jQuery 1.11.3"
              -- Right-click on that link and select "Save Link As..."
              -- You will be prompted to save jquery-1.11.3.min.map
              -- Save the file to your c:\vault\jquery\
              
          3) Now, in your c:\vault\jquery\, you should see these 3 files		    
    			jquery-1.11.2.js
    			jquery-1.11.2.min.js
    			jquery-1.11.2.min.map
		  
	c) Create a directory WAR/jquery
	   Right-click on WAR -> new -> folder
	   Folder name:  jquery
	   
	d) Drag and drop the 3 files from c:\vault\jquery\ to your WAR/jquery
			jquery-1.11.2.js
			jquery-1.11.2.min.js
			jquery-1.11.2.min.map
	

	
	
5) Add another controller called "WelcomeController2"

   -- It will by default forward to welcome2.jsp

   package stuff.ab;

	import net.sourceforge.stripes.action.DefaultHandler;
	import net.sourceforge.stripes.action.ForwardResolution;
	import net.sourceforge.stripes.action.Resolution;
	 
	public class MainPageController2 extends BaseActionBean
	{    

		@DefaultHandler   
		public Resolution showMain()
		{     
			logger.debug("showMain() started");
			return new ForwardResolution("/WEB-INF/jsp/main2.jsp");   
		}
	}
				
   
   
6) Add a new jsp called main2.jsp



<%@ page contentType="text/html;charset=ISO-8859-1" language="java" %>
<%@ taglib prefix="s" uri="http://stripes.sourceforge.net/stripes.tld" %>
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core"%>
<c:set var="contextPath" value="${pageContext.request.contextPath}"/>


<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"    "http://www.w3.org/TR/html4/strict.dtd" >

 
 <html>  
   <head>  
     <title>main2</title>  
     
     <meta name="viewport" content="width=device-width, initial-scale=1.0">  
     <!-- Bootstrap -->  
     <link href="${contextPath}/lib/bootstrap-3.3.4/dist/css/bootstrap.min.css" rel="stylesheet" media="screen">  
   
     <!-- Bootstrap CSS Themes -->  
     <link href="${contextPath}/lib/bootstrap-3.3.4/dist/css/bootstrap-theme.min.css" rel="stylesheet" media="screen">  

     <!-- HTML5 shim and Respond.js IE8 support of HTML5 elements and media queries -->  
     <!--[if lt IE 9]>  
       <script src="${contextPath}/lib/bootstrap-3.3.4/assets/html5shiv.js"></script>  
       <script src="${contextPath}/lib/bootstrap-3.3.4/assets/respond.min.js"></script>  
     <![endif]-->  
   </head>  
   
   
   
   
   <body>  
     <h1>Hello, world!</h1>  
   
   
   
   
   
   
     <!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->  
     <script src="${contextPath}/lib/jquery/jquery-1.11.2.min.js"></script>  
     
     <!-- Include all compiled plugins (below), or include individual files as needed -->  
     <script src="${contextPath}/lib/bootstrap-3.3.4/dist/js/bootstrap.min.js"></script>  
   </body>  
 </html>
