How to Use Curl with PKI Cert
-----------------------------

Assumptions
 A) You have a PKI client cert
 B) You want to use it with curl
 C) You have CYGWIN installed (to use the curl command
 
Procedures
----------
 1. Download and install openssl
 
 
 2. Convert your .p12 file format to a .pem
    a. Open a DOS window by pressing <Start><Run>CMD
    b. In the DOS window, type-in this:
       CMD> openssl pkcs12 -in c:\temp\myClientCert.p12 -out c:\temp\myClientCert.pem -nodes
             
            NOTE:  the -nodes option removes the password
            

3. Get the RSA private key from your .pem signed cert
   CMD> openssl rsa -in c:\temp\myClientCert.pem -out c:\temp\myClientCertKey.pem
        -- Generates this file:  myClientCertKey.pem
        
        
4. Use curl with the signed cert (in pem format) and RSA private key (in pem format)
   CYGWIN> cd /cygdrive/c/temp
   CYGWIN> curl --key myClientCertKey.pem  --cert myClientCert.pem 'https://myWebServer.com/whatever' --insecure
