How to Get Hard Drive Speed in Windows
--------------------------------------

References
----------
https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2012-R2-and-2012/cc742157(v=ws.11)?redirectedfrom=MSDN



Procedure
---------
 1. Open a CMD window in administrator mode
 
 2. Run this command to get speed info of your C:\
    DOS> winsat disk -drive C:
 
 3. Run this command to get speed info of your D:\
    DOS> winsat disk -drive D:
