How to Change Outlook Email rows and columns for the inbox view
----------------------------------------------------------------

References
----------
https://guides.downstate.edu/c.php?g=654922&p=4860853


Procedure
---------
 1. Pull View -> View Settings -> Other Settings
 
 2. In the "Other Settings" popup
    a. Click on "Column Font..."
       Select a different size column font
       
    b. Click on "Row Font..."
       Select a different size row font
