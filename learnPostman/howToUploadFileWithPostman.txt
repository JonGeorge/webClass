How to Upload a File with Postman
---------------------------------
Problem:  You want to simulate uploading a file to a REST point
          but the front-end does not exist

Solution: Upload a file using Postman


Procedure
---------
 1. Startup Postman
 
 2. Start a new request
    a. Set Request type to POST
    b. Click on Body -> form-data
	c. Click in the key field (when it's empty)
    -- Select the File dropdown option
    -- Set the key=file
