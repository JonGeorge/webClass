How to Use Maven to Package the JavaFX App
------------------------------------------
The development team is using Linux or Windows and all have Oracle Java JDK 1.8
The production users will have an Oracle Java JRE 1.8
-- I want my development team to use Maven to package up the app
-- I do not want my production users to have to install anything


Procedure
---------
 1. Build a small maven JavaFX app
 
 
 2. Package the javaFX app
 
 
 3. Deploy it to a Windows box and try to run it
