How to Install JavaFX
---------------------
JavaFX does not come with the Java JDK.  It must be installed separately.

From Oracle Java 8 until Oracle Java 10, JavaFX was included
From Oracle Java 11 on, JavaFX was not included
The OpenJDK does not cme with JavaFX


Procedure
---------
 1. If you have OpenJDK installed, then you must install Oracle Java 8, 9, or 10
    (so you can use it locally)
 2. Make sure your target user has Oracle Java 8, 9, or 10 installed as well
