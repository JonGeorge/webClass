How to Create a Parser using ANTLR 4
------------------------------------

Assumptions:
 A) You have a Windows 7 workstation or later
 B) You have a Java JDK installed
 
 
 
Procedures
----------
 1. Follow the procedures in howToCreateJavaCommandLineProgramUsingIntellijMaven.txt
    Assumption:  The intellij project is called queryParser


 2. Add this dependency to your pom.xml
 
     <dependency>
        <groupId>org.antlr</groupId>
        <artifactId>antlr4-runtime</artifactId>
        <version>4.3</version>
     </dependency>
     
     
     
 3. Add this plugin to your <build><plugins>...</plugins></build> section of your pom.xml
    [so mvn will compile the parser in addition to any Java code]
    
      <plugin>
        <groupId>org.antlr</groupId>
        <artifactId>antlr4-maven-plugin</artifactId>
        <version>4.3</version>
        <executions>
          <execution>
            <goals>
              <goal>antlr4</goal>
            </goals>
            <configuration>
              <!-- Generate parse tree visitor interface and base class. -->
              <visitor>true</visitor>
            </configuration>
          </execution>
        </executions>
      </plugin>

      
 4. Add a simple HelloWorld.g4 grammar
    a. Create this directory:  /src/main/antlr4
        
    b. Create this file:  HelloWorld.g4  (in the /src/main/altr4/ directory)
    
		grammar Hello;              // Define a grammar called Hello
		r  : 'hello' ID;            // match keyword hello followed by an identifier
		ID : [a-z]+;                // match lower-case identifiers
		WS : [ \t\r\n]+ -> skip;    // skip spaces, tabs, newline, \r
   
   
   NOTE:  If you create your Hello.g4 file under /src/main/antlr4/com/stuff
          Then, your generated class will exist under the com.stuff package
          
   
 
 5. Compile your code
    unix> cd ~/intellijProjects/queryParser
    unix> mvn clean compile
    
    --- You should now see ./target/classes/HelloParser.class
    
 
 6. Tell Intellij to look for the generated antlr classes
    a. Pull File / Project Structure
    b. Click Modules  [on the left]
    c. In the directory,
         Right-click on target/generated-sources/antlr4 -> Sources
